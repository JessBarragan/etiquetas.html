# NOMBRE PARA QUE SIRVE
1 HTML Delimita y engloba toda la página web, que consta de cabecera y cuerpo.
2 HEAD Delimita y engloba la cabecera de una página
3 TITLE Dentro de (HEAD), se muestra en la barra del título de la ventana del navegador.
4 BODY Delimita y engloba el cuerpo de la página
5 H1 , H2, H3..... Párrafos que son encabezamientos (con distintos niveles).
6 P Párrafos de texto normales.
7 P ALING ='"CENTER" El atributo align permite alinear el texto del párrafo.
8 BR Permite partir un párrafo empezando una línea nueva pero sin dejar espacio.
9 HR Pone una linea horizontal de separación. (admite atributos)
10 BLOCKQUOTE Sangría el párrafo.
11 BASE Define la URL base para las URLs relativas en la página.
12 LINK Usada para enlazar JavaScript y CSS externos con el documento HTML .
13 META Define los metadatos que no pueden ser definidos usando otro elemento.
14 STYLE Etiqueta de estilo usada para escribir CSS en línea.
15 SCRIPT Define ya sea un script interno o un enlace hacia un script externo.
16 NOSCRIPT Define un contenido alternativo, cuando el navegador no soporta scripting.
17 SECTION Define una sección en un documento.
18 NAV Define una sección que solamente contiene enlaces de navegación
19 ARTICLE Define contenido que podría existir independientemente del resto.
20 ASIDE Define los contenidos vagamente relacionados con el resto del contenido.
21 HEADER Define la cabecera de una página o sección (logotipo, el titulo del sitio, o tabla).
22 FOOTER Define el pie de una página o sección.
23 ADDRESS Define una sección que contiene información de contacto.
24 MAIN Define el contenido principal o importante en el documento.
25 PRE Indica que el contenido esta preformateado y que debe ser preservado.
26 OL Define una lista ordenada de artículos.
27 UL Define una lista de artículos sin orden.
28 LI Define un artículo de una lista enumerada
29 DL Define una lista de definiciones, es decir, una lista de términos
30 DT Representa un término definido por el siguiente
.
31 DD Representa la definición de los términos listados antes que él.
32 FIGURE Representa una figura ilustrada como parte del documento.
33 FIGCAPTION Representa la leyenda de una figura.
34 DIV Representa un contenedor genérico sin ningún significado especial.
35 A Representa un hiperenlace, enlazando a otro recurso.
36 EM Representa un texto enfatizado, como un acento de intensidad.
37 STRONG Representa un texto especialmente importante.
38 S Representa contenido que ya no es exacto o relevante.
39 CITE Representa el título de una obra.
40 CODE Representa un código de ordenador.
41 VAR Representa a una variable (expresión matemática o contexto de programación)
42 SAMP Representa la salida de un programa o un ordenador.
43 SUB Representan un subíndice y un superíndice, respectivamente.
44 I Representa un texto en una voz o estado de ánimo alterno
45 B Representa un texto hacia el cual se utiliza para propósitos utilitarios.
46 MARK Representa texto resaltado con propósitos de referencia,
47 SPAN Representa texto sin un significado específico.
48 IMG Representa una imagen.
49 AUDIO Representa un sonido o stream de audio.
50 VIDEO Representa un video, y sus archivos de audio y capciones asociadas,
